  Pour le SMS (Africa's Talking — recommandé pour l'Afrique de l'Ouest) :
  1. SMS_ENABLED=true
  2. AT_API_KEY → clé API depuis https://africastalking.com
  3. AT_USERNAME → ton nom d'utilisateur AT (souvent le nom de ton app)

  Pour WhatsApp (Twilio — plus simple à démarrer) :
  1. WA_ENABLED=true
  2. TWILIO_ACCOUNT_SID + TWILIO_AUTH_TOKEN → depuis https://console.twilio.com
  3. TWILIO_WA_FROM → le numéro WhatsApp Twilio (sandbox: +14155238886)

  À ne pas oublier :
  - CRON_SECRET → une chaîne aléatoire pour sécuriser les endpoints cron (ex:
  génère avec openssl rand -hex 32)
  - DEFAULT_PHONE_PREFIX → 228 si Togo, 229 si Bénin, 225 si CIV

  Les seuils d'alertes (retard, inactivité, etc.) sont tous optionnels — les
  valeurs par défaut commentées s'appliquent si tu ne les mets pas.
